Politique de Confidentialité

Invoice Pro - Votre Confidentialité Compte

Dernière Mise à Jour : Janvier 2025
Date d'Entrée en Vigueur : Janvier 2025
Notre Engagement : Invoice Pro est conçu avec la confidentialité comme principe fondamental. Vos données restent sur votre appareil. Nous ne collectons, ne stockons ni ne partageons vos informations personnelles sur aucun serveur.

1. Introduction

Bienvenue sur Invoice Pro (« nous », « notre » ou « l'Application »). Cette Politique de Confidentialité explique comment Invoice Pro gère vos informations lorsque vous utilisez notre application mobile disponible sur les plateformes iOS et Android.

Invoice Pro est un générateur de factures hors ligne conçu pour les freelances et les petites entreprises. Nous nous engageons à protéger votre confidentialité et à être transparents sur nos pratiques en matière de données.

2. Informations que nous NE collectons PAS

Invoice Pro est conçu avec la confidentialité dès la conception. Nous voulons être clairs sur ce que nous NE collectons PAS :

  • Pas de Comptes Utilisateur : Nous n'exigeons ni ne stockons de comptes utilisateur, noms d'utilisateur, mots de passe ou identifiants de connexion
  • Pas de Stockage Cloud : Nous ne téléchargeons pas vos factures, données clients ou informations commerciales sur des serveurs
  • Pas de Suivi Personnel : Nous ne suivons pas votre localisation, vos habitudes de navigation ou vos modèles d'utilisation de l'application
  • Pas d'Analytique : Nous n'utilisons pas de services d'analytique pour surveiller votre comportement dans l'application
  • Pas de Données Publicitaires : Nous ne collectons pas de données à des fins publicitaires
  • Pas de Partage avec des Tiers : Nous ne partageons pas vos données avec des tiers (sauf indication contraire dans la Section 3)

3. Comment Vos Données Sont Stockées

Stockage Local sur l'Appareil

Toutes vos données sont stockées localement sur votre appareil en utilisant un stockage local sécurisé (AsyncStorage sur React Native). Cela inclut :

  • Votre profil d'entreprise (nom, e-mail, téléphone, adresse, logo)
  • Informations sur les clients (noms, coordonnées)
  • Données de facturation (articles, montants, dates, statut)
  • Paramètres et préférences de l'application
  • Images et logos téléchargés

Important : Vos données ne quittent jamais votre appareil sauf si vous choisissez explicitement de les exporter ou de les partager.

Contrôle des Données

Vous avez un contrôle total sur vos données :

  • Vous pouvez exporter toutes vos données à tout moment en utilisant la fonction d'exportation intégrée
  • Vous pouvez supprimer toutes les données depuis les paramètres de l'application
  • La désinstallation de l'application supprimera définitivement toutes les données stockées localement

4. Achats In-App (RevenueCat)

Invoice Pro propose des modèles premium via un achat in-app unique. Nous utilisons RevenueCat, un service tiers, pour traiter les achats.

Ce que RevenueCat Collecte :

  • Identifiant utilisateur anonyme (généré par RevenueCat)
  • Historique des achats et statut d'abonnement
  • Type d'appareil et plateforme (iOS/Android)

RevenueCat ne reçoit ni ne stocke :

  • Vos factures ou données commerciales
  • Informations sur les clients
  • Tout contenu que vous créez dans l'application

Politique de Confidentialité de RevenueCat : https://www.revenuecat.com/privacy

Les informations de paiement (détails de carte bancaire) sont traitées par Apple App Store ou Google Play Store et ne sont jamais partagées avec nous ou RevenueCat.

5. Autorisations que Nous Demandons

Invoice Pro peut demander les autorisations suivantes pour fonctionner correctement :

Accès à la Bibliothèque de Photos (Optionnel)

Objectif : Vous permettre de télécharger le logo de votre entreprise ou de vos clients
Quand : Uniquement lorsque vous choisissez d'ajouter une image
Stockage : Les images sont stockées localement sur votre appareil

Accès au Système de Fichiers (Optionnel)

Objectif : Exporter/importer des données et enregistrer des factures PDF
Quand : Lorsque vous exportez des données ou enregistrez une facture
Contrôle : Vous choisissez où les fichiers sont enregistrés

Note : Toutes les autorisations sont optionnelles. Vous pouvez utiliser l'application sans accorder ces autorisations, bien que certaines fonctionnalités puissent être limitées.

6. Partage et Exportation de Données

Invoice Pro vous permet de partager volontairement des données :

Génération et Partage de PDF

  • Lorsque vous générez un PDF de facture, il est créé localement sur votre appareil
  • Vous pouvez partager des PDF par e-mail, applications de messagerie ou enregistrer sur votre appareil
  • Ce partage est entièrement contrôlé par vous via les fonctionnalités de partage natives de votre appareil

Exportation de Données

  • Vous pouvez exporter toutes vos données sous forme de fichier JSON
  • Les fichiers d'exportation sont enregistrés localement ou partagés à votre discrétion
  • Nous ne recevons ni ne stockons aucune donnée exportée

7. Confidentialité des Enfants

Invoice Pro n'est pas destiné aux enfants de moins de 13 ans (ou l'âge applicable du consentement numérique dans votre juridiction). Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants.

Si vous pensez qu'un enfant a utilisé notre application, veuillez nous contacter à golukicoding@gmail.com et nous vous fournirons des conseils sur la suppression des données.

8. Vos Droits à la Confidentialité

Selon votre localisation, vous pouvez avoir les droits suivants :

Droits RGPD (Utilisateurs Européens)

  • Droit d'Accès : Toutes vos données sont accessibles dans l'application
  • Droit de Suppression : Supprimez toutes les données via les paramètres de l'application
  • Droit à la Portabilité : Exportez vos données à tout moment
  • Droit d'Opposition : Vous contrôlez toute collecte de données

Droits CCPA (Utilisateurs Californiens)

  • Droit de Savoir : Cette politique explique toutes les pratiques en matière de données
  • Droit de Suppression : Supprimez les données à tout moment via les paramètres de l'application
  • Droit de Refus : Aucune donnée n'est vendue ou partagée

Étant donné que toutes les données sont stockées localement, vous avez un contrôle direct sur vos informations sans avoir besoin de nous soumettre des demandes.

9. Sécurité

Nous prenons la sécurité au sérieux :

  • Stockage Local : Les données sont stockées en utilisant des mécanismes de stockage d'appareil sécurisés fournis par iOS et Android
  • Pas de Transmission Réseau : Vos données commerciales ne sont jamais transmises sur Internet
  • Sécurité de l'Appareil : Vos données sont protégées par les fonctionnalités de sécurité de votre appareil (code d'accès, authentification biométrique, chiffrement)

Votre Responsabilité : Gardez votre appareil sécurisé avec un code d'accès fort et activez le chiffrement de l'appareil. Si votre appareil est perdu ou volé, vos données peuvent être accessibles par d'autres.

10. Utilisateurs Internationaux

Invoice Pro est disponible dans le monde entier et prend en charge 7 langues. Étant donné que toutes les données sont stockées localement sur votre appareil :

  • Aucune donnée n'est transférée au-delà des frontières
  • Les données ne sont pas soumises aux lois de juridiction étrangère
  • Vos données restent sous votre juridiction locale

  • L'utilisation continue de l'application après les modifications constitue une acceptation

Nous vous encourageons à consulter régulièrement cette politique.
